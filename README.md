# Personal Portfolio

This is my personal portfolio page, created to present who I am, my current focus as a programming student, and my main contact links.

The project was built with HTML and CSS, focusing on a clean visual structure, responsive design, smooth navigation between sections, and a simple but elegant interface.

## About the Project

This portfolio was created as part of my learning journey in front-end development.  
The main goal is to practice page structure, styling, responsiveness, layout organization, and visual presentation using only front-end technologies.

The page includes an introduction section, an about section, and a contact section with links to my social media profiles and email.

## Features

- Personal introduction section
- About me section
- Contact section
- Social media links
- Email contact link
- Responsive layout
- Smooth scrolling
- Scroll snap between sections
- Background image with overlay
- Hover animations
- Clean and organized design

## Technologies Used

- HTML
- CSS
